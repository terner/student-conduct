## ภาพรวมระบบ

ระบบคะแนนความประพฤตินักเรียน สำหรับโรงเรียนไทย
- นักเรียนเริ่มต้นที่ 100 คะแนนต่อปีการศึกษา
- หักคะแนนเมื่อทำผิด เพิ่มคะแนนเมื่อทำความดี
- แจ้งเตือนเมื่อถึงเกณฑ์ ทำทัณฑ์บน ออกรายงาน

## 3 บทบาทหลัก

| บทบาท | สิทธิ์ |
|--------|-------|
| **ผู้ดูแลสูงสุด (superadmin)** | ทุกอย่าง |
| **ผู้ดูแลระบบ (admin)** | จัดการข้อมูลโรงเรียน, อนุมัติคะแนน, ดูรายงาน, ตั้งค่าระบบ |
| **ครู (teacher)** | บันทึกคะแนน, ดูรายงานห้องตัวเอง |
| **นักเรียน (student)** | ดูคะแนนตัวเอง |
